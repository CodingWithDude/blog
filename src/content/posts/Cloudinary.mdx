## Why I Started Using Cloudinary

As an aspiring self taught developer I am always excited to discover tools that make building great projects a bit easier. The first thing that I loved was I now only had one domain and one service to store my images and videos.

```js title="next.config.js" {4} showLineNumbers
const nextConfig = {
  reactStrictMode: true,
  swcMinify: true,
  images: { domains: ['res.cloudinary.com'] },
};
```

<ExternalLink href="https://cloudinary.com/">Cloudinary</ExternalLink> also allows
you to automatically scale down your images right in the url request so the size
you need is the size you request.

```js title="GradientBackground.tsx" showLineNumbers /h_500/
// Tailwindcss Background image
<div
  className="h-full transition-all ease-in
    bg-[url('https://res.cloudinary.com/image/upload/h_500/blog/bg_gradient.png')]
    bg-top bg-no-repeat opacity-[0.2] will-change-transform"
></div>

// Image
<img src="https://res.cloudinary.com/image/upload/h_500/blog/bg_gradient.png" alt="image" />
```

All developers have access to a generous [free account](https://cloudinary.com/users/register_free) to build side projects and test implementations of the wide feature set Cloudinary provides.

## Transformations

With the Transformations Cookbook in your Cloudinary Dashboard you have access to a host of different image transformations you can apply on the fly just like we did with scaling image size. Here are just few examples of the types of transformations built in.

- ## Face detection-based image cropping

  <div className="flex flex-col items-center sm:flex-row gap-6 pb-6">

  With Cloudinary, cropping an image can be easily done on-the-fly in the
  cloud. Cloudinary can automatically detect faces' position within an image,
  and crop a thumbnail based on the detected face.

  <img
    src="https://res.cloudinary.com/demo/image/upload/w_200,h_200,c_thumb,g_faces/w_200,f_auto,q_auto/couple.jpg"
    alt="faces"
    className="rounded-2xl w-[150px] h-[150px]"
  />

  </div>

  ```js
  title="SampleImage.tsx" showLineNumbers /w_150,h_150,c_thumb,g_faces/
  <img
    src="https://res.cloudinary.com/demo/image/upload/w_150,h_150,c_thumb,g_faces/w_200,f_auto,q_auto/couple.jpg"
    alt="Faces"
  />
  ```

- ## Overlay an image over detected faces

  <div className="flex flex-col items-center sm:flex-row gap-6 pb-6">

  <img
    src="https://res.cloudinary.com/demo/image/upload/l_badge,g_faces,w_1.0,h_1.0,fl_region_relative/c_pad,ar_1.0/v1603370517/Spain_national_football_team_Euro_2012_final.jpg"
    alt="faces"
    className="rounded-2xl w-[150px] h-[150px]"
  />

  Cloudinary automatically detects and searches for faces within an image. You can overlay another image of your choice on top of those detected faces.

  </div>
